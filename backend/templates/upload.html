Upload your sales CSV or Excel file to power the entire dashboard.

Select a .csv or .xlsx file from your device.

Required File Format

Your file must contain the following 8 columns exactly as named below. Column order does not matter.

Column Name Type Description Example
Order_ID Number Unique identifier for each order 1001
Order_Date Date Date the order was placed — format must be YYYY-MM-DD 2024-03-15
Product_Name Text Name of the product sold Laptop
Category Text Product category grouping Electronics
Quantity Number Number of units sold in the order 2
Selling_Price Number Price per unit at which the product was sold 45000
Cost_Price Number Cost per unit to acquire or produce the product 30000
Region Text Geographic region where the order was placed Mumbai
